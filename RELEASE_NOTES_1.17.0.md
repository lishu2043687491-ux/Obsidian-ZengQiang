# ZengQiang Enhanced 1.17.0

## What changed

- Reorganized the settings center by real work scenarios: editing and layout, knowledge organization, content and assets, automation and AI, and third-party plugin management.
- Added a clear English overview for community users while retaining the Chinese guide.
- Updated the public manifest description and added a complete GPL-3.0-only license file for community review.

## Privacy and release scope

- Release assets contain only `main.js`, `manifest.json`, and `styles.css`.
- No personal notes, templates, settings, caches, backups, absolute paths, cookies, tokens, or API keys are included.
- Vault and clipboard access are limited to user-triggered features described in the README.
