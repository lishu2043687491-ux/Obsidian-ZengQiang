# ZengQiang Enhanced 1.16.6

## 本版定位与介绍更新

1.16.6 是一次社区展示与文档收口版本，代码功能不变，重点是把插件公开介绍为：**Obsidian 增强办公体验工具**，面向飞书、企微等办公工具迁移用户。

插件上传名称仍为英文 **ZengQiang Enhanced**；中文名称只用于公开介绍和插件详情描述，不写入 `manifest.name`。

## 社区介绍补充

- 补充浮动 T 工具栏、6 页设置中心和统一入口定位。
- 详细说明模板库的分组、回收、三处入口同步与个人模板隔离。
- 详细说明表格三模式：成熟版、礼书版、全关，以及原生 Markdown 数据边界。
- 补充 OneNote 富文本粘贴、图片/附件本地化、宽页面、复制成图和短链接能力。
- 补充 Claudian 跨设备存档、候选排除、Agent 接入修复和视频时间轴预览。
- 明确外部图片工具属于可选/测试能力，上传和删除均需用户主动操作。
- 更新隐私、公开发行包范围、安装方式和单插件使用说明。

## 合规

- `manifest.json` 简介改为面向目标人群的中文描述，并以中文句号结尾。
- 版本关系：本地真相源、公开仓库与 Release 均为 `1.16.6`。
- Release 附件仍只包含 `main.js`、`manifest.json`、`styles.css`。
- 不发布 `data.json`、个人模板、笔记、缓存、备份、本地路径或密钥。
