# OneNote 剪贴板 HTML 样本

从 OneNote 复制后，在 Obsidian 弹窗点 **「导出原始 HTML（调试样本）」**，将剪贴板内容保存为本目录下的 `.html` 文件。

## 建议采集的笔记页

| 文件名建议 | 来源 |
|---|---|
| `cognitive-dissonance.html` | 可迁移 / 认知失调 |
| `wechat-merchant-ops.html` | 业务中台 / 3.1微信收款商户-运维 |
| `behavior-skills.html` | 212思维力 / 3、行为技能 |
| `llm-checklist.html` | 大语言模型自检清单（三列表中表） |

验收：`npm run test:paste`
