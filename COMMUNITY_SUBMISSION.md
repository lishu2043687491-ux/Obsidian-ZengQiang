# Obsidian Community Plugin Submission

Single listing: **Obsidian增强体验** (`feishu-doc-toolbar`).

Repository: https://github.com/lishu2043687491-ux/Obsidian-ZengQiang

## Prerequisites

- [ ] `README.md`, `LICENSE`, `manifest.json` at repository root
- [ ] GitHub Release tag **`1.0.0`** (no `v` prefix)
- [ ] Release assets: `main.js`, `manifest.json`, `styles.css`
- [ ] Companion `markdown-table-enhancer` documented in README (`companion-plugins/`)

## Developer Dashboard

1. Sign in at https://community.obsidian.md
2. Connect GitHub account (same account that owns the repo)
3. **Plugins → New plugin** → `https://github.com/lishu2043687491-ux/Obsidian-ZengQiang`
4. Submit → automated review
5. In-app listing within ~24h after approval

## Review checklist

- Command ids do not include the plugin id prefix
- `description` ≤ 250 characters, ends with `.`, no emoji
- No user `data.json` or private vault content in the repo
- README states companion table plugin install from `companion-plugins/markdown-table-enhancer/`

## Subsequent releases

New GitHub Release tag must match `manifest.json` `version`.
